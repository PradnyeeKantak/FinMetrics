## About The Project
You can calculate the Value-at-Risk measure using historical stock price data from a CSV file. The data get arranged in order and the value corresponsing to the required percentile is obtained, according to the determined confidence level.
<p align="right">(<a href="#readme-top">back to top</a>)</p>

### Built With
Python 3.10
<p align="right">(<a href="#readme-top">back to top</a>)</p>

<!-- GETTING STARTED -->
## Getting Started

Get historcial stock prices in CSV format - data on each stock must be in a separate file.
Points to remember:
* Whether the column names in the CSV file are the same as those used in the code ("High Price", "Low Price", etc.) - if not, they will have to be changed accordingly.
* The "filename" that the user inputs is the complete path with extension.
* Enter the filename as "End" when you are done with all the files, in order to get a dictionary listing the VaR measures for all given files.

### Prerequisites
Python editor (Jupyter notebook/ Spyder) or Anaconda

### Installation
Download var_calculator.py
